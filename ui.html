User Interface

Here are some of the app's core screens and functionalities:

Users can create accounts to log in to the app.
Home Screen to navigate to different parts of the app.
Users can input personal information to receive personalized recommendations.
Users can input meals to track their calorie intake. They can also see visualizations of these metrics.
Users can store and manipulate difference recipes.
Users can add ingredients to their shopping list.
